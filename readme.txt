WP Category Thumbnail create a thumbnail widget

== Description ==

This plugin provides thumbnails for categories fetched from the latest page/post of the corresponding category. The post thumbnails are generated from its featured image in the post.

== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory
2. Activate the plugin

= Features =

* Show thumbnails for latest page/post of the corresponding category
